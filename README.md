# 📊 CryptoInsight — Dashboard de Análise de Criptomoedas

CryptoInsight é um painel web responsivo para análise em tempo real das principais criptomoedas do mercado. Desenvolvido com foco em performance, integração com APIs e visualização de dados.

## 🚀 Visão Geral

Objetivos do projeto:

- Exibir dados de criptomoedas em tempo real
- Permitir análises visuais através de gráficos interativos
- Oferecer filtros dinâmicos e busca avançada
- Demonstrar boas práticas em frontend moderno

## 🛠️ Tecnologias Utilizadas

- **React + Vite** — Estrutura leve e rápida
- **Zustand** — Gerenciamento de estado
- **Axios** — Consumo de APIs REST
- **Tailwind CSS** — Estilização utilitária
- **Recharts** — Visualização de dados
- **CoinGecko API** — Fonte de dados inicial

## 🌟 Sobre o Projeto

### 🧠 Por que o projeto CryptoInsight é promissor?

CryptoInsight nasce em um momento onde a informação financeira precisa ser acessível, visual e confiável. O mercado de criptomoedas está em constante movimento — e acompanhar essas mudanças com clareza e simplicidade ainda é um desafio.

### 🎯 Qual é o propósito da aplicação?

O CryptoInsight foi criado para:

- 📊 Facilitar o entendimento do mercado de criptomoedas, mesmo para quem não é especialista
- 🛠️ Demonstrar um alto nível técnico em React, Zustand, consumo de APIs e gráficos em tempo real
- 🌐 Servir como uma ferramenta aberta e útil para a comunidade

## 🧭 Como funciona?

**Para leigos e técnicos:**

1. A pessoa acessa o dashboard de qualquer dispositivo (celular, tablet, desktop)
2. A aplicação se conecta a APIs públicas confiáveis (como CoinGecko, Binance ou Coinbase)
3. Busca as informações mais recentes sobre as criptomoedas mais relevantes
4. Esses dados são exibidos em gráficos interativos, tabelas e indicadores
5. O usuário pode pesquisar, filtrar e acompanhar mudanças em tempo real
6. Tudo com uma interface clara, rápida e leve
